Azure Services Guide

Networking Services

  • API Apps: Easily build and consume cloud APIs.

  • API Management: Publish APIs securely and at scale.

Example Azure CLI command for creating a virtual network:

az network vnet create / 
            --name MyVNet /
            --resource-group MyResourceGroup /
            --location eastus /
            --address-prefixes 10.0.0.0/16

Compute Services

  • Virtual Machines: Deploy Windows or Linux VMs.

  • Azure Kubernetes Service (AKS): Manage containerized applications.

Example Azure CLI command for creating an AKS cluster:

az aks create /
            --resource-group MyResourceGroup /
            --name MyAKSCluster /
            --node-count 3 /
            --enable-addons monitoring /
            --generate-ssh-keys

Data and Storage Services

  • Azure SQL Database: Managed relational database service.

  • Azure Cosmos DB: Globally distributed NoSQL database.

Example Azure CLI command for creating an Azure SQL Database:

az sql server create /
            --resource-group MyResourceGroup /
            --name MySqlServer /
            --location eastus /
            --admin-user myadmin /
            --admin-password MyP@ssw0rd123

AI and Machine Learning Services

  • Azure Machine Learning: Build, deploy, and manage ML models.

  • Azure Cognitive Services: APIs for vision, speech, language, and more.

Example Azure CLI command for creating an Azure Machine Learning workspace:

az ml workspace create /
            --resource-group MyResourceGroup /
            --workspace-name MyMLWorkspace

Security and Identity Services

  • Azure Active Directory: Identity and access management.

  • Azure Key Vault: Securely manage keys, secrets, and certificates.

Example Azure CLI command for creating an Azure Key Vault:

az keyvault create /
            --resource-group MyResourceGroup /
            --name MyKeyVault

Remember that this guide provides only a glimpse of Azure's extensive services. For more details, explore the official Azure documentation.
